# Simon Game

A simple browser-based Simon game built with HTML, CSS, and JavaScript.  
Test your memory by repeating the sequence of colors generated by the game!

## Game Rules

1. The game will show a sequence of colors.
2. Click the buttons in the same order.
3. If you click the wrong button, the game ends.
4. Try to reach the highest level possible!

## How to Play

1. Open `index.html` in your browser.
2. Press any key to start the game.
3. Watch the sequence of flashing buttons and listen to their sounds.
4. Repeat the sequence by clicking the buttons in order.
5. The game will increase the sequence by one color each level.

## Technologies Used

- HTML
- CSS
- JavaScript
- jQuery (for DOM manipulation)

## How to Run Locally

1. Clone the repository or download the files.
2. Make sure the `sounds/` folder is in the same directory as `index.html`.
3. Open `index.html` in your browser.
4. Press any key to start the game and have fun!
